This extension is a shortcut to navigate to any website. Type in the address (ncluding the http://), and press Enter/click the button next to the box.

The Premiere extension panel is based on the Chrome browser, so everything that works in Chrome should work in this extension as well. The main caveat is that there is no support for tabs, so if a website tries to open a page in a new tab, nothing will happen.

If you need to navigate back to this page, to load a different website, right click on the page and select "back" in the contextual menu

Click the Star button to access a pre-selected list of favourites (customisation coming soon)
